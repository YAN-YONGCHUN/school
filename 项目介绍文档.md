# 《攀峰计划》项目介绍文档

## 一、项目名称

攀峰计划：校园计算机学科文化与创新实践平台

## 二、项目定位

《攀峰计划》是一套围绕“计算机学科 + 校园文化 + 创新实践”打造的纯前端网页设计作品。项目从校园真实学习与活动场景出发，以网页设计为载体，把学科展示、课程体系、实验空间、成果荣誉、登录注册、创新社区、智慧服务和数据驾驶舱整合为一个完整平台。

本作品既适合作为网页设计大赛参赛作品，也具备继续作为学院宣传、社团招募、活动展示和作品汇报页面长期使用的潜力。

## 三、设计思路

### 1. 主题升级

原始项目主题偏向登山平台，与计算机学科关联较弱。本次重构保留“攀登”的精神意象，将其升级为“攀登代码高峰、攀登创新高地”的表达，使主题既积极向上，又与计算机专业高度契合。

### 2. 学科与校园文化融合

项目不仅展示技术内容，还强调校园文化氛围：

- 课堂学习与实验空间联动
- 技术社团、开源共学和活动沙龙
- 校园数字服务与真实项目实践
- 师生协同、朋辈互助和成果传播

这样使网页既有专业性，也有校园温度。

### 3. 多页叙事结构

整站采用多页结构，形成完整叙事链：

1. 首页：总览主题与平台价值
2. 学科全景：展示专业方向与成长路径
3. 实验空间：展现实践场景与实验室文化
4. 课程图谱：体现课程体系与项目制学习
5. 协同育人：展示校内外协同与真实项目来源
6. 成果荣誉：呈现作品成果与文化故事
7. 创新社区：提供报名交互入口
8. 智慧服务：提供资源中心与服务模块
9. 登录注册：展示纯前端账户系统
10. 账户中心：展示个人资料与报名记录
11. 数据驾驶舱：展示本地数据联动与统计看板

## 四、视觉设计亮点

### 1. 统一视觉系统

全站使用统一的颜色体系、卡片组件、区块标题、按钮与导航样式，保证不同页面之间风格统一、识别度高。

### 2. 主题表达鲜明

主色采用深青绿色与暖橙色组合：

- 深青绿色体现科技感、理性和专业气质
- 暖橙色体现青春、活力与创新热情

结合柔和浅色背景和网格纹理，营造兼具数字感与校园温度的视觉氛围。

### 3. 轻量化表达

页面主要通过 CSS 渐变、卡片层次、排版和交互细节构建质感，不依赖大图和复杂框架，保证体积轻巧。

## 五、技术亮点

### 1. 纯前端静态化实现

项目采用 HTML + CSS + JavaScript 构建，无需后端即可运行，适合 GitHub Pages 静态部署。

### 2. 本地数据驱动

通过 `localStorage` 管理账户信息、社区报名和数据看板，实现纯前端环境下的完整交互闭环。

### 3. 登录注册与账户中心

新增纯前端登录注册界面和账户中心页面，使整站不再只是介绍网站，而更像可使用的校园数字平台。

### 4. 前台与看板联动

用户在创新社区提交报名后，数据会实时反映到数据驾驶舱中，方便答辩时进行交互演示。

### 5. 品牌信息集中可替换

学校名称、学院名称、比赛名称和 Logo 字样均集中配置，方便快速修改为真实学校信息。

## 六、功能说明

### 1. 首页

展示平台主题、学校品牌信息、核心价值、动态数据与快速搜索功能。

### 2. 学科全景

介绍计算机学科主要方向、培养逻辑和成长路径。

### 3. 实验空间

展示实验室、开放活动和实践空间文化。

### 4. 课程图谱

展示课程分层结构、学年任务与项目制学习案例。

### 5. 协同育人

展示师生协同、校企合作与真实需求转化路径。

### 6. 成果荣誉

展示成果类型、校园故事和数据化荣誉墙。

### 7. 创新社区

支持填写报名表单并查看最新报名动态。

### 8. 智慧服务

展示资源中心、学科日历与服务模块。

### 9. 登录注册

支持纯前端账户登录与注册。

### 10. 账户中心

支持查看个人信息、编辑资料和查看个人报名记录。

### 11. 数据驾驶舱

展示报名统计、方向热度与报名记录，体现作品的可运行性。

## 七、部署优势

- 纯前端项目
- 不需要服务器
- 资源体积轻
- 适合比赛在线展示
- 维护成本低

## 八、项目价值总结

《攀峰计划》不只是一个介绍型网页，而是一套具有展示力、交互性、延展性和静态可部署性的校园数字平台作品。它将计算机学科的专业特征与校园文化氛围融合起来，用网页设计语言讲述“技术如何在校园中生长”的故事。
