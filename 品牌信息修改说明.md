# 品牌信息修改说明

如果你要把作品改成自己学校的版本，主要改下面这一个文件：

## 一、需要修改的文件

文件位置：

`js/site-data.js`

## 二、重点修改位置

打开 `js/site-data.js`，最上面有这一段：

```js
campus: {
    schoolName: "星河大学",
    collegeName: "计算机与人工智能学院",
    contestName: "校园网页设计大赛参赛作品",
    logoText: "CX",
    location: "中国 · 智慧校园示范区"
}
```

你只需要改这几个值：

- `schoolName`
  这里改成你的学校名称

- `collegeName`
  这里改成你的学院名称

- `contestName`
  这里改成比赛名称或比赛展示标题

- `logoText`
  这里是页面左上角 Logo 里的字母，建议改成学校或学院缩写

- `location`
  这里可以改成学校所在城市、校区或你想展示的描述

## 三、举例

如果你是“XX大学 计算机学院”，可以改成：

```js
campus: {
    schoolName: "XX大学",
    collegeName: "计算机学院",
    contestName: "第X届校园网页设计大赛参赛作品",
    logoText: "CS",
    location: "中国 · XX市"
}
```

## 四、Logo 怎么处理

目前项目使用的是文字型 Logo，也就是：

- 不需要额外图片
- 不会增加项目体积
- 更适合比赛快速替换

如果你后面想换成图片 Logo，也可以继续改，但目前这版为了轻量和统一风格，先用文字 Logo。

## 五、哪些页面会自动同步

你改完 `js/site-data.js` 之后，这些地方会自动同步变化：

- 顶部品牌名称
- 学校名称
- 学院名称
- 比赛名称
- Logo 字样
- 首页和部分页面中的学校信息

也就是说，你不用一页一页手动改。
